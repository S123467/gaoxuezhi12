## 功能模块说明

### 模块一：高血脂关键风险因素识别

**算法**: Lasso-Logistic 回归、XGBoost-SHAP、随机森林交叉验证

**功能**:
1. 使用Lasso回归进行特征初筛，剔除多重共线性
2. 使用Logistic回归进行线性主效应推断，计算OR值
3. 使用XGBoost+SHAP挖掘非线性特征重要性和交互机制
4. 使用随机森林进行交叉验证，多模型共识得到核心特征
5. 输出特征权重排序和重要性可视化

**论文结论实现**: TG、TC、LDL-C、HDL-C 及血尿酸构成高血脂风险识别的主要代谢主轴；痰湿质与BMI的交互放大效应尤为显著。

### 模块二：三级风险分层预警

**算法**: CART决策树、中西医融合三级标签（低/中/高风险）

**功能**:
1. 基于论文提取的关键阈值生成先验标签：
   - 痰湿积分 ≈ 60 分
   - TG ≈ 1.7 mmol/L
   - TC ≈ 6.2 mmol/L
   - 活动能力 ≈ 40 分
2. 使用CART决策树学习显性阈值规则
3. 输出三级风险预警：低风险/中风险/高风险
4. 自动识别并列出异常风险因素

**论文性能**: CART模型 5折交叉验证准确率达到 **95.00%**

### 模块三：个体化干预方案优化

**算法**: 画像分型（代谢异常/肥胖并发/功能受限）+ 约束组合优化

**功能**:
1. 根据用户特征自动进行画像分型：
   - **代谢异常主导型**: 血脂异常为主，体质和活动能力尚可 → 优先强化调理
   - **肥胖并发型**: BMI超标 + 痰湿质 + 代谢异常 → 调理 + 运动联合干预
   - **功能受限型**: 活动能力下降 + 高龄 + 多种并发症 → 低强度稳步改善方案
2. 根据分型应用对应约束
3. 生成所有满足预算约束的可行组合
4. 在约束下最大化预期改善效果，输出最优方案和备选方案
5. 约束条件：6个月总成本不超过2000元

**输出**: 调理等级、运动强度、运动频率、总成本、预期效果

## 核心算法描述

### 三级风险分层算法

```
输入: 用户特征向量 X = [痰湿积分, TG, TC, HDL-C, LDL-C, 血尿酸, BMI, 活动能力, 年龄]
过程:
  1. 对每个指标，按照论文提取的阈值判断异常程度
  2. 累积风险分数，结合CART决策树预测
  3. 输出风险等级: 低(0) / 中(1) / 高(2)
输出: 风险等级 + 异常因素列表
```

关键阈值完全来自论文CART决策树提取结果，保证算法与论文一致。

### 干预组合优化算法

```
目标函数: max 总预期效果 = 6 × (调理月效果 + 运动月效果)
约束条件:
  1. 6 × (调理月成本 + 运动月成本) ≤ 预算上限
  2. 满足患者分型的强度约束（如最大运动强度限制）
输出: 按效果排序的前N个可行方案，最优方案排在第一位
```

## 创新点

1. **中西医融合创新**: 首次将中医体质积分与西医生化指标融合进行高血脂风险评估，充分发挥中西医结合优势，提高了痰湿体质人群的风险识别准确率

2. **可解释性风险分层**: 基于CART决策树提取显性阈值规则，每一级风险的判定依据清晰可解释，便于医护人员理解和使用

3. **约束优化个性化推荐**: 在预算和身体耐受度双重约束下，为不同分型患者推荐效果最优的干预组合方案，平衡干预效果、经济成本和身体耐受度

4. **完整闭环系统**: 从特征筛选 → 风险预警 → 患者分型 → 干预推荐形成完整的决策支持闭环
