# 高血脂风险分层预警与干预推荐系统 - 前端

## 说明

前端采用简单的 HTML + Bootstrap 5 + 原生 JavaScript 实现，满足：

1. 四个核心页面：信息录入 → 风险计算 → 风险结果 → 干预推荐
2. 前后端分离，通过 REST API 与后端通信
3. 响应式设计，美观清爽，直接在浏览器打开即可使用

## 文件结构

```
frontend/
├── index.html       # 主页面（包含四个步骤）
├── about.html       # 关于页面
└── README.md        # 说明文件
```

## 使用方法

1. 确保后端服务已启动在 `http://localhost:5000`
2. 直接用浏览器打开 `index.html`
3. 在页面顶部填写后端API地址（默认就是 `http://localhost:5000/api`）
4. 填写用户信息，点击下一步即可

## 页面流程

| 步骤 | 页面 | 功能 |
|------|------|------|
| 1 | 信息录入 | 输入年龄、性别、身高、体重、血脂指标、痰湿积分、活动能力、预算 |
| 2 | 风险计算 | 调用后端API，异步计算结果 |
| 3 | 风险结果 | 展示风险等级、异常因素、患者分型 |
| 4 | 干预推荐 | 展示最优方案和备选方案，包括成本与预期效果 |

## 前后端接口

前端调用 `POST /api/predict` 接口，传入用户信息，后端返回完整结果。
