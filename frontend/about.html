关于本系统

项目背景

本系统基于数模竞赛论文《基于中西医融合信息的高血脂风险识别、分层预警与干预优化研究》（队伍编号: CMC2604725）的研究成果开发， 将论文中的数学模型完整实现为可运行的软件系统，用于高血脂风险评估和个体化干预推荐。

核心算法（完全复现论文）

模块 算法 论文结论
关键风险因素识别 Lasso-Logistic + XGBoost-SHAP + 随机森林交叉验证 TG、TC、LDL-C、HDL-C、血尿酸是主要代谢主轴，痰湿质与BMI有交互放大效应
三级风险分层预警 CART决策树 + 中西医融合先验 5折交叉验证准确率 95.00%，关键阈值: 痰湿≈60分、TG≈1.7、TC≈6.2、活动能力≈40
个体化干预优化 画像分型 + 约束组合优化 不同分型患者推荐不同策略，6个月总成本≤2000元

患者分型与推荐策略

患者类型 特征 推荐策略
代谢异常主导型 血脂异常为主，体质和活动能力尚可 优先强化调理
肥胖并发型 BMI超标 + 痰湿质 + 代谢异常 调理 + 运动联合干预
功能受限型 活动能力下降 + 高龄 + 多种并发症 低强度稳步改善方案

系统架构

  • 后端: Python Flask，完整实现论文三大算法模块，提供REST API
  • 前端: HTML + Bootstrap 5 + 原生JavaScript，前后端分离
  • 机器学习: Scikit-learn + XGBoost + SHAP
